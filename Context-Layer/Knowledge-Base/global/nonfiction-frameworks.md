# Khung Cấu trúc Viết Phi hư cấu

## Frameworks phổ biến

### 1. Problem-Solution
```
Vấn đề → Hậu quả → Giải pháp → Evidence → CTA
```
Dùng cho: bài blog, case study, content marketing.

### 2. Pyramid (MECE)
```
Big Idea → Supporting Points → Evidence
```
Dùng cho: tài liệu chuyên môn, báo cáo, consulting.

### 3. Before-After-Bridge
```
Trước (pain) → Sau (desired state) → Cầu nối (how)
```
Dùng cho: sales copy, landing page, social content thuyết phục.

### 4. Chronological / Process
```
Bước 1 → Bước 2 → ... → Kết quả
```
Dùng cho: hướng dẫn, tutorial, handbook.

### 5. Compare-Contrast
```
Option A vs Option B → Criteria → Verdict
```
Dùng cho: review, đánh giá, tư vấn lựa chọn.

### 6. Storytelling (STAR)
```
Situation → Task → Action → Result
```
Dùng cho: case study, personal branding, ebook.

### 7. Inverted Pyramid (Journalism)
```
Kết luận → Chi tiết quan trọng → Background
```
Dùng cho: tin tức, summary, briefing.

## Chọn framework theo mục đích

| Mục đích | Framework đề xuất |
|----------|------------------|
| Thuyết phục | Problem-Solution, Before-After-Bridge |
| Giáo dục | Pyramid, Chronological |
| Kể chuyện | STAR, Storytelling |
| So sánh | Compare-Contrast |
| Tóm tắt | Inverted Pyramid |
| Tài liệu | Pyramid, Chronological |
