# Cấu trúc Kịch bản Video

## Short-form (15-60s)

### Structure: Hook-Value-CTA
```
HOOK (0-3s) → Giữ viewer, gây tò mò
VALUE (4-X) → Deliver promise, insights
CTA (cuối)  → Hành động: follow, share, comment
```

### Hook types
1. **Question hook:** "Tại sao 90% startup fail?"
2. **Stat hook:** "73% Gen Z mua hàng vì content"
3. **Controversy:** "Đây là sai lầm lớn nhất khi..."
4. **Story:** "Tháng trước, tôi suýt phá sản vì..."
5. **Demonstration:** Bắt đầu bằng kết quả wow

### Retention techniques
- **Pattern interrupt:** Đổi cảnh, zoom, text overlay mỗi 3-5s
- **Open loop:** Hứa reveal gì đó ở cuối
- **Cliffhanger:** "Nhưng điều xảy ra tiếp theo..."
- **List promise:** "3 cách [X]" — viewer chờ hết list

## Long-form (3-15 phút)

### Structure: Hook-Intro-Body-Recap-CTA
```
HOOK (0-15s)    → Attention grabber
INTRO (15-60s)  → Context, promise
BODY (bulk)     → Deliver value, 3-7 sections
RECAP (30s)     → Tóm tắt key points
CTA (15-30s)    → Subscribe, next video, resource
```

## Spoken-style writing rules

1. **Viết cho tai, không cho mắt** — đọc to phải tự nhiên
2. **Câu ngắn** — 5-12 từ/câu optimal cho nói
3. **Tránh mệnh đề phụ** — khó nói, khó nghe
4. **Dùng từ ngữ đời thường** — "sợ" thay vì "lo ngại"
5. **Nghỉ (pause)** — ghi [beat] hoặc [...] cho chỗ ngừng
