# Knowledge: Intake Department

## Mẫu Phân tách Brief

| Loại brief | Signals | Cần extract |
|------------|---------|-------------|
| Ebook request | "viết ebook", "viết sách", "5 chương" | Chapters, audience, depth |
| Social request | "viết bài Facebook", "content MXH" | Platform, tone, CTA |
| Video request | "kịch bản", "script", "video ngắn" | Duration, style |
| Edit request | "sửa", "review", "polish" | Bản gốc, vấn đề cụ thể |
| Grade request | "chấm", "đánh giá", "score" | Bài cần chấm, rubric |

## Phân loại Mục đích Nội dung

```
User Intent
├── Educate     → ebook, tài liệu, explainer
├── Persuade    → sales page, landing copy, pitch
├── Entertain   → social, storytelling, viral
├── Inform      → report, whitepaper, analysis
├── Convert     → CTA-driven, funnel content
└── Critique    → review, grade, feedback
```

## Quy tắc Ánh xạ Đối tượng

| Audience signal | Profile | Tone recommendation |
|----------------|---------|-------------------|
| "cho người mới" | Beginner | Friendly, simple |
| "chuyên gia", "B2B" | Expert | Professional, data-driven |
| "Gen Z", "MXH" | Youth | Casual, trendy |
| "doanh nhân", "CEO" | Executive | Concise, strategic |
| "nhân viên nội bộ" | Internal | Direct, actionable |

## Mẫu Mơ hồ thường gặp

Khi brief mơ hồ, hỏi lại:

1. **Thiếu audience** → "Bài viết này cho ai đọc?"
2. **Thiếu mục đích** → "Sau khi đọc xong, bạn muốn người đọc làm gì?"
3. **Thiếu format** → "Output dạng ebook hay bài social?"
4. **Thiếu scope** → "Bao dài? 500 từ hay 5000 từ?"
5. **Thiếu tone** → "Giọng chuyên nghiệp hay thân thiện?"
