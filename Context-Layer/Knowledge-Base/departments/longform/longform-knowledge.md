# Knowledge: Longform Department

## Mẫu Cấu trúc Ebook

| Chapters | Suitable for | Pattern |
|----------|-------------|---------|
| 3 chương | Short guide, quick start | Intro → Core → Apply |
| 5 chương | Standard ebook | Why → What → How → Case → Next |
| 7-10 chương | Comprehensive guide | Module-based, independent chapters |
| 10+ chương | Reference book | Part-based, with index & glossary |

## Luồng Logic Chương

```
Chương 1: BỐI CẢNH — Vì sao đọc cuốn này?
    → Pain point, trend, hoặc câu hỏi lớn
    
Chương 2-3: KIẾN THỨC NỀN — Hiểu cơ bản
    → Concept, framework, terminology
    
Chương 4-6: THỰC HÀNH — Làm thế nào?
    → Step-by-step, tools, examples
    
Chương 7-8: CASE STUDY — Ai đã làm?
    → Real examples, lessons learned
    
Chương cuối: NEXT STEPS — Rồi sao tiếp?
    → Action items, resources, CTA
```

## Nguyên tắc Rõ ràng Giáo dục

1. **Concept → Example → Application** — không lý thuyết suông
2. **1 section = 1 idea** — đọc heading biết nội dung
3. **Progressive complexity** — đơn giản → phức tạp
4. **Recall triggers** — cuối chapter có tóm tắt key points
5. **Visual thinking** — bảng, sơ đồ, comparison khi cần
