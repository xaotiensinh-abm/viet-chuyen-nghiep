# Knowledge: Video Department

## Cấu trúc Video ngắn

### 15s Script (TikTok/Reels)
```
[0-3s] HOOK — 1 câu gây tò mò
[3-10s] CONTENT — 1 ý chính, ví dụ nhanh
[10-15s] CTA — action rõ ràng
```

### 30s Script  
```
[0-3s] HOOK — stat/question/controversial
[3-12s] CONTEXT — setup vấn đề
[12-25s] PAYOFF — answer/insight/solution
[25-30s] CTA
```

### 60s Script
```
[0-3s] HOOK
[3-10s] SETUP — bối cảnh + vì sao quan trọng
[10-30s] BODY — 2-3 points chính
[30-50s] EXAMPLES — minh họa cụ thể
[50-57s] SUMMARY — 1 câu kết
[57-60s] CTA
```

## Nguyên tắc Giữ chân

1. **3-second rule** — hook phải work trong 3s
2. **Pattern interrupt** — đổi visual/audio mỗi 3-5s
3. **Open loop** — gợi mở ở đầu, trả lời ở cuối
4. **List promise** — "3 điều..." → viewer đợi hết list
5. **Payoff must deliver** — không clickbait rồi giải thích khác
