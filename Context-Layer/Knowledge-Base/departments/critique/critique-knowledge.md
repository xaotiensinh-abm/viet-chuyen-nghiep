# Knowledge: Critique Department

## Argument Analysis Framework

### CLAIM-EVIDENCE-REASONING (CER)

```
Claim: [Khẳng định của bài viết]
    │
    ├── Evidence: [Bằng chứng nào support?]
    │   ├── Is it relevant? (liên quan?)
    │   ├── Is it sufficient? (đủ?)
    │   └── Is it current? (cập nhật?)
    │
    └── Reasoning: [Logic nối claim với evidence?]
        ├── Is it valid? (logic hợp lệ?)
        ├── Is it sound? (tiền đề đúng?)
        └── Are there hidden assumptions?
```

### Chấm điểm Calibration Examples

| Band | Score | Đặc điểm | Ví dụ tình huống |
|------|-------|----------|----------------|
| 🟢 Xuất sắc | 90+ | Evidence strong, logic clear, style polished | Bài có data, source, structure rõ |
| 🔵 Tốt | 75-89 | Mostly solid, minor gaps | Ý tốt nhưng thiếu 1-2 sources |
| 🟡 Khá | 60-74 | Readable but weak areas | Structure OK, evidence thin |
| 🟠 TB | 40-59 | Major issues | Logic gaps, AI-sounding |
| 🔴 Yếu | <40 | Fundamental problems | Off-topic, fabricated, incoherent |

## Quy tắc Viết Phản hồi

- **CỤ THỂ** — "paragraph 3 thiếu evidence" > "viết chưa tốt"
- **CONSTRUCTIVE** — chỉ lỗi + gợi ý hướng sửa
- **PRIORITIZED** — nêu issue quan trọng nhất trước
- **BALANCED** — có điểm mạnh + điểm yếu
