# Bộ nhớ Kinh nghiệm: Ban Chiến lược

## Mẫu chọn góc tiếp cận đã kiểm chứng

### Pattern 1: Contrarian angle (highest engagement)
```
Topic: "Remote work"
Obvious: "5 lợi ích của remote work"
Contrarian: "3 điều doanh nghiệp VN hiểu SAI về remote work"
→ Result: 3x engagement vì challenge assumptions
```

### Pattern 2: Data-anchor angle (highest credibility)
```
Topic: "AI trong giáo dục"
Weak: "AI sẽ thay đổi giáo dục"
Data-anchored: "73% trường ĐH VN chưa có policy AI — đây là vấn đề"
→ Result: Specific number → credibility + hook
```

### Pattern 3: Personal-story angle (highest relatability)
```
Topic: "Quản lý thời gian"
Generic: "10 cách quản lý thời gian hiệu quả"
Personal: "Tôi thử 10 phương pháp quản lý thời gian — chỉ 3 cái work"
→ Result: Personal voice → anti-AI + relatable
```

## Mẫu kiến trúc Outline

| Content type | Outline pattern | Example |
|-------------|----------------|---------|
| **Listicle** | N items + intro + conclusion | "7 tools AI cho marketer VN" |
| **Problem-Solution** | Pain → Analysis → Solution → CTA | "Tại sao email marketing VN fail" |
| **Story-Lesson** | Hook → Context → Conflict → Resolution → Takeaway | "Bài học từ thất bại startup AI" |
| **Framework** | Concept → Components → How-to → Examples | "Framework 3C cho content VN" |
| **Comparison** | A vs B → Criteria → Deep dive → Verdict | "ChatGPT vs Gemini cho content VN" |

## Công thức Ý tưởng lớn

1. **Unexpected truth**: "[Topic] không phải [common belief], mà là [surprise]"
2. **Hidden cost**: "Cái giá thật của [popular thing] mà ít ai nói"
3. **Simple framework**: "[Complex topic] chỉ cần [N] bước"
4. **Counter-example**: "Tại sao [best practice] không work ở VN"
