# Curriculum Framework — CoreModule

## ADDIE Model cho Thiết Kế Giáo Trình

### 5 Phases
```
Analyze → Design → Develop → Implement → Evaluate
   │         │         │          │            │
   │         │         │          │            └─ Measure learning outcomes
   │         │         │          └─ Delivery & facilitation  
   │         │         └─ Content creation + exercises
   │         └─ Module outline + Bloom mapping
   └─ Audience + needs + gap analysis
```

## Bloom's Taxonomy — Quick Reference

### Verb Bank theo Level

| Level | Loại | Verbs (Tiếng Việt) | Verbs (English) |
|-------|------|--------------------|--------------------|
| 1 | Nhớ (Remember) | Liệt kê, kể tên, nhận diện, xác định | List, name, identify, define |
| 2 | Hiểu (Understand) | Giải thích, mô tả, so sánh, tóm tắt | Explain, describe, compare, summarize |
| 3 | Áp dụng (Apply) | Sử dụng, thực hiện, áp dụng, tạo | Use, execute, apply, create |
| 4 | Phân tích (Analyze) | Phân tích, so sánh, đánh giá, phân biệt | Analyze, compare, evaluate, distinguish |
| 5 | Đánh giá (Evaluate) | Đánh giá, phê bình, biện hộ, khuyến nghị | Evaluate, critique, justify, recommend |
| 6 | Sáng tạo (Create) | Thiết kế, xây dựng, phát triển, đề xuất | Design, build, develop, propose |

### Learning Objective Formula
```
Sau [module/khóa học] này, học viên sẽ có thể:
[Bloom verb] + [nội dung cụ thể] + [điều kiện/context]

Ví dụ:
✅ "Sử dụng ChatGPT để viết email marketing cho sản phẩm B2B"
✅ "Phân tích hiệu quả của 3 chiến dịch AI marketing thực tế"
❌ "Hiểu về AI" (quá mơ hồ)
❌ "Biết cách dùng tool" (không đo được)
```

## Module Structure Standards

### Thời lượng đề xuất
| Loại module | Thời lượng | Số exercises |
|------------|-----------|-------------|
| Micro-lesson | 15-30 phút | 1 |
| Standard module | 45-90 phút | 2-3 |
| Deep dive | 2-3 giờ | 3-5 |
| Workshop session | 3-4 giờ | 5-8 |

### Content Ratio
```
40% Lý thuyết (concepts, examples, case studies)
40% Thực hành (exercises, projects, labs)
20% Assessment (quiz, reflection, review)
```

### Progression Rules
```
Module N prerequisites = Module N-1 outcomes
Bloom level N ≤ Bloom level N-1 + 1
Each module ends with bridge to next
Capstone = combines all previous modules
```

## Assessment Matrix

### Formative (trong quá trình học)
| Type | Timing | Purpose |
|------|--------|---------|
| Quick quiz | Giữa module | Check hiểu cơ bản |
| Reflection | Cuối module | Self-assessment |
| Peer discussion | Sau case study | Collaborative learning |

### Summative (đánh giá chính thức)
| Type | Timing | Purpose |
|------|--------|---------|
| Module test | Cuối module | Verify mastery |
| Phase project | Cuối phase (3-4 modules) | Apply combined skills |
| Capstone project | Cuối khóa | Prove overall competency |

## Khóa Học Type Templates

### Online Course (Self-paced)
```
5-12 modules × 45-60 phút
Format: Video + text + quiz
Delivery: LMS (Udemy, Teachable, Notion)
Certificate: completion-based
```

### Workshop / Bootcamp
```
2-5 ngày × 3-4 giờ/ngày
Format: Live + exercises + group work
Delivery: Zoom / on-site
Certificate: project-based
```

### Corporate Training
```
1-3 ngày × full-day
Format: Slides + hands-on + SOP
Delivery: On-site / hybrid
Assessment: KPI-based (30/60/90 day)
```
